# Ecommerce Website

A simple and responsive e-commerce website built using **HTML, CSS, and JavaScript**.  
This project focuses on front-end development concepts such as layout design, user interaction, and cart management.

## Features

- Product listing with images, names, and prices  
- Add to cart functionality  
- Dynamic cart update with total price calculation  
- Checkout form to collect user details  
- Clean and modern user interface with a green theme  
- Responsive layout for better user experience  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript
